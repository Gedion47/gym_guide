# Gym Guide

Gym Guide is a flutter app that runs both on ios and android. The project includes the following pages:

    - Splash Screen
    - Login Page
    - Signup Page
    - Training Page
    - Workout Page
    - Progress Tracking Page
    - Diet Plan page
    - Profile page
    - Payment Simulation

## Getting Started

## Installation

**Step 1:**

Download or clone this repo by using the link below:

```
https://github.com/Gedion47/gym_guide.git
```

**Step 2:**

Go to project root folder

```
cd gym_guide
```

**Step 3:**

Execute the following command in console to get the required dependencies:

```
flutter pub get
```

**Step 4:**

Make sure you have a device (Simulator/Emulator/Actual Mobile) connected then run the app:

```
flutter run
```
